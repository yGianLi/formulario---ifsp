## 📌 Sobre o Projeto

Este projeto é um formulário de pesquisa de satisfação desenvolvido como atividade acadêmica, com o objetivo de coletar feedback sobre a experiência dos usuários no IFSP.

A aplicação permite avaliar aspectos como qualidade de ensino, infraestrutura e interesses acadêmicos, além de coletar informações básicas dos participantes.

## 🧩 Funcionalidades

### 👤 Informações Pessoais

* Nome completo (obrigatório)
* E-mail (obrigatório)
* Idade (validação entre 15 e 99 anos)
* Cidade e Estado

### 🏫 Vínculo com a Instituição

* Tipo de vínculo (estudante, professor, ex-aluno, outro)
* Curso
* Período

### ⭐ Avaliação

* Qualidade do ensino
* Infraestrutura
* Recomendação da instituição

### 📚 Interesses Acadêmicos

Seleção de áreas como:

* Front-end
* Back-end
* Full Stack
* Java
* JavaScript
* Python
* C/C++
* Banco de Dados
* Inteligência Artificial
* Segurança da Informação

### 💬 Feedback

* Campo aberto para comentários e sugestões
